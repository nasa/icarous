\doxysubsection{Detailed Description}
This class captures a single position represented in either Euclidean or Lat/\+Lon coordinates. ~\newline
 This class is immutable.

This class is designed to be resilient; The methods do not return errors when a position in the \char`\"{}wrong\char`\"{} geometry is provided. So the method \mbox{\hyperlink{classlarcfm_1_1_position_a9cfdb3d5df2f27253744bf0047075408}{x()}} will return a value even when the original position was provided in \mbox{\hyperlink{classlarcfm_1_1_lat_lon_alt}{Lat\+Lon\+Alt}}. ~\newline
 The correspondence is as follows\+: 
\begin{DoxyItemize}
\item latitude corresponds to Y 
\item longitude corresponds to X 
\item altitude corresponds to altitude (obviously) 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
This class represents a point or a 3-\/dimensional vector. This class contains methods to make the printing of \mbox{\hyperlink{classlarcfm_1_1_vect3}{Vect3}}\textquotesingle{}s be more appropriate for a position (units of length), instead of the abstract \mbox{\hyperlink{classlarcfm_1_1_vect3}{Vect3}} which has no natural units. ~\newline

This parses a space or comma-\/separated string as a \mbox{\hyperlink{classlarcfm_1_1_point}{Point}} (an inverse to the to\+String method). If three bare values are present, then it is interpreted as the default units for a point\+: \mbox{[}NM,NM,ft\mbox{]}. If there are 3 value/unit pairs then each values is interpreted with regard to the appropriate unit. If the string cannot be parsed, an INVALID value is returned.

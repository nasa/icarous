\doxysubsection{Detailed Description}
The Class \mbox{\hyperlink{classlarcfm_1_1_c_d_s_s}{CDSS}}. This is a dedicated class that only uses the \mbox{\hyperlink{classlarcfm_1_1_c_d3_d}{CD3D}} (\mbox{\hyperlink{classlarcfm_1_1_c_d_cylinder}{CDCylinder}}) detection algorithm. The more general form is \mbox{\hyperlink{classlarcfm_1_1_c_d_s_s_core}{CDSSCore}}, which may utilize arbitrary \mbox{\hyperlink{classlarcfm_1_1_detection3_d}{Detection3D}} detection algorithms.

Usage\+: \mbox{\hyperlink{classlarcfm_1_1_c_d_s_s}{CDSS}} cdss = new CDSS(); cdss.\+set\+Distance(5,\char`\"{}\+NM\char`\"{}) cdss.\+set\+Height(950,\char`\"{}ft\char`\"{}) boolean det = cdss.\+detection(so,vo,si,vi,\+T);

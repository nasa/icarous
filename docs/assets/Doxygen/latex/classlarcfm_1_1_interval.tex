\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlarcfm_1_1_interval}{Interval}} represents a interval of double\textquotesingle{}s from a lower bound to an upper bound. This class is immutable.

Whether the interval is interpreted as open or closed is context-\/dependent. There are various membership tests allowing for the different interpretations.

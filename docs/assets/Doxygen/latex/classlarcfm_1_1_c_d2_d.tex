\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlarcfm_1_1_c_d2_d}{CD2D}} is an algorithm for 2-\/D conflict {\itshape detection}.

All units in this file are {\itshape internal}\+: 
\begin{DoxyItemize}
\item \mbox{\hyperlink{classlarcfm_1_1_units}{Units}} of distance are denoted \mbox{[}d\mbox{]} 
\item \mbox{\hyperlink{classlarcfm_1_1_units}{Units}} of time are denoted \mbox{[}t\mbox{]} 
\item \mbox{\hyperlink{classlarcfm_1_1_units}{Units}} of speed are denoted \mbox{[}d/t\mbox{]} 
\end{DoxyItemize}

REMARK\+: X points to East, Y points to North.

Naming Convention~\newline
 The intruder is fixed at the origin of the coordinate system. 
\begin{DoxyItemize}
\item D \+: Diameter of the protected zone \mbox{[}d\mbox{]} 
\item B \+: Lower bound of lookahed time interval \mbox{[}t\mbox{]} (B \texorpdfstring{$>$}{>}= 0) 
\item T \+: Upper bound of lookahead time interval \mbox{[}t\mbox{]} (B \texorpdfstring{$<$}{<} T) 
\item s \+: Relative 2-\/D position of the ownship \mbox{[}d,d\mbox{]} 
\item vo \+: Ownship velocity vector \mbox{[}d/t,d/t\mbox{]} 
\item vi \+: Traffic velocity vector \mbox{[}d/t,d/t\mbox{]} 
\end{DoxyItemize}

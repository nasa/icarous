\doxysubsection{Detailed Description}
Encapsulates a geometric polygon. The polygon is defined in terms of its vertex coordinates. This implementation assumes a simply connected polygon. The Polygon is otherwise quite general allowing multiply connected regions. The class provides a containment test for points and uses bounding rectangles to speed up computations. ~\newline


Note\+: It does not use the \char`\"{}\+Bounding\+Rectangle\char`\"{} class to avoid importing Lat/\+Lon classes such as \mbox{\hyperlink{classlarcfm_1_1_great_circle}{Great\+Circle}}.

\doxysubsection{Detailed Description}
Polygon detection. Based on UNVERIFIED PVS code.

A \mbox{\hyperlink{classlarcfm_1_1_moving_polygon2_d}{Moving\+Polygon2D}} contains a list of \mbox{\hyperlink{classlarcfm_1_1_vect2}{Vect2}} vertices and a corresponding list of velocities. These lists must be the same size.

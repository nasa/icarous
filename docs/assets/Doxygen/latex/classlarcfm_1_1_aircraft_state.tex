\doxysubsection{Detailed Description}
This class captures a limited history of an aircraft\textquotesingle{}s position and velocity. ~\newline
 This class essentially operates as an array, where the time field determines the order. It is intended that this method is called frequently and the oldest values are dropped. The maximum number of values stored is available through max\+Buffer\+Size(). The oldest value (that is, the smallest t) is always in element 0, the latest element (that is, the largest time) is always at index \mbox{\hyperlink{classlarcfm_1_1_aircraft_state_a89b08a958ae14fb52637ca87706f69fd}{size()}} -\/ 1.

The primary accessors are \mbox{\hyperlink{classlarcfm_1_1_aircraft_state_aea538edb532477053eb9f59aa7deaf7c}{position()}} and \mbox{\hyperlink{classlarcfm_1_1_aircraft_state_a43694b43006e8380e9b9c83037d17360}{velocity()}} which are indexed from 0 to \mbox{\hyperlink{classlarcfm_1_1_aircraft_state_a89b08a958ae14fb52637ca87706f69fd}{size()}}-\/1. The latest values are located at \mbox{\hyperlink{classlarcfm_1_1_aircraft_state_a89b08a958ae14fb52637ca87706f69fd}{size()}}-\/1.

The method \mbox{\hyperlink{classlarcfm_1_1_aircraft_state_a1ce86b67bc55d527035805a336025c74}{get()}} will return a Euclidean projection of the position and velocity given the projection function in \mbox{\hyperlink{classlarcfm_1_1_aircraft_state_a0ba0bea029c7901e417d1a46e36be0ea}{set\+Projection()}} or \mbox{\hyperlink{classlarcfm_1_1_aircraft_state_ae4c4f0e56d7db0d18b5e3cc9bd251233}{update\+Projection()}}. Be careful\+: if you do not specifically indicate which projection to use, this class will make a choice for you, that is probably not what you expect.

In addition, this method can predict future points (currently this uses a quadratic regression approximation). These predictions are available through the methods pred(t), and pred\+Linear(t). These are based on the Euclidean projected points.

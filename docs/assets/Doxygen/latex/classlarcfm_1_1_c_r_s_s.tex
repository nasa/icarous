\doxysubsection{Detailed Description}
Conflict resolution.

Conflict resolution between an ownship and traffic aircraft using state information.

Possible resolution values\+: 
\begin{DoxyItemize}
\item LoS -- Loss of separation resolution 
\item None -- No resolution available 
\item Conflict -- Conflict resolution 
\item Unnecessary -- Conflict resolution is not needed 
\end{DoxyItemize}

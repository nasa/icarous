\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlarcfm_1_1_safety_buffers}{Safety\+Buffers}} class. This computes the psi value (increase in safety buffer size) needed to preserve a given certainty for detection readings, based on relative distance and speed, and data accuracy values.

In general, these functions will fail (give a negative value) if invalid NIC/\+NAC values are given, or if a NIC/\+NAC corresponds to an unknown accuracy category for that dimension.

Calculations are based on A. Narkawicz, C. Munoz, H. Herencia-\/\+Zapana, G. Hagen. \char`\"{}\+Formal Verification of Safety Buffers for State-\/\+Based Conflict Detection.\char`\"{}

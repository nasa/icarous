\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlarcfm_1_1_c_r3_d}{CR3D}} is an algorithm for 3-\/D conflict {\itshape resolution} that provides avoidance maneuvers for the ownship

Unit Convention~\newline


All units in this file are {\itshape internal}\+: 
\begin{DoxyItemize}
\item \mbox{\hyperlink{classlarcfm_1_1_units}{Units}} of distance are denoted \mbox{[}d\mbox{]} 
\item \mbox{\hyperlink{classlarcfm_1_1_units}{Units}} of time are denoted \mbox{[}t\mbox{]} 
\item \mbox{\hyperlink{classlarcfm_1_1_units}{Units}} of speed are denoted \mbox{[}d/t\mbox{]} 
\item \mbox{\hyperlink{classlarcfm_1_1_units}{Units}} of turn are radians \mbox{[}rad\mbox{]} 
\end{DoxyItemize}

REMARK\+: X points to East, Y points to North. Angles are in True North/clockwise convention.

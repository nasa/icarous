\doxysubsection{Detailed Description}
This class creates a local Euclidean projection around a given point. This projection may be used to transform geodetic coordinates (\mbox{\hyperlink{classlarcfm_1_1_lat_lon_alt}{Lat\+Lon\+Alt}} objects) into this Euclidean frame, using the \mbox{\hyperlink{classlarcfm_1_1_orthographic_projection_a30dc0e38059f07d0317a7ea6407dec48}{project()}} method. Also points within this frame, may be found in geodetic coordinates with the \mbox{\hyperlink{classlarcfm_1_1_orthographic_projection_ac0089b5c96de556a968e35dfc6066512}{inverse()}} method. As long as the points are close to the projection point, the errors will be very small.

This is a standard orthographic projection, and is functionally similar to the ENU projection (it uses different calculations). ~\newline
 It distorts distances from the tangent point, compressing them more as the distance increases (similar to a picture of a planet from distant space). ~\newline
 Note that this projection should be considered to have a hard limit of \mbox{\hyperlink{classlarcfm_1_1_great_circle_a93bff9d2b9cbee7e8ee230208057002c}{Great\+Circle.\+spherical\+\_\+earth\+\_\+radius}} as the maximum allowed distance from the tangent reference point, and significant distortion may occur at much smaller distances.

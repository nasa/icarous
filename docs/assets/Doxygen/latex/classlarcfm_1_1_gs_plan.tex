\doxysubsection{Detailed Description}
\mbox{\hyperlink{classlarcfm_1_1_gs_plan}{Gs\+Plan}} -- an alternate yet convenient way to store a linear plan. There are no times stored. Instead the ground speed at each point is specified.

Since there are no times in a \mbox{\hyperlink{classlarcfm_1_1_gs_plan}{Gs\+Plan}}, only the order of the points is maintained. This often eliminates the need to continually recalculate times. After an appropriate \mbox{\hyperlink{classlarcfm_1_1_gs_plan}{Gs\+Plan}} is constructed it can be converted to a linear plan using the \char`\"{}linear\+Plan\char`\"{} method

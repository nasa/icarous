\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classlarcfm_1_1_interval_set}{Interval\+Set}} class represents a set of \char`\"{}double\char`\"{} values. Ranges of doubles are maintained as intervals (\mbox{\hyperlink{classlarcfm_1_1_interval}{Interval}}). These Intervals are ordered consecutively from lowest to highest. Standard set operations of {\ttfamily in} (membership), {\ttfamily union} (set union), {\ttfamily intersect} (set intersection), and {\ttfamily diff} (set difference) are provided.

Within the \mbox{\hyperlink{classlarcfm_1_1_interval_set}{Interval\+Set}}, intervals are generally considered closed (including end-\/points), and the results off operations are closed intervals. This implies that the interval difference is between a set of closed intervals and one or more open intervals.

The intervals are numbered 0 to \mbox{\hyperlink{classlarcfm_1_1_interval_set_abbe3e787fda6e93212e0f655b647de00}{size()}}-\/1. To cycle through the intervals one may\+:


\begin{DoxyPre}{\ttfamily 
\mbox{\hyperlink{classlarcfm_1_1_interval_set}{IntervalSet}} set;

for( int i = 0; i < set.size(); i++) \{
  \mbox{\hyperlink{classlarcfm_1_1_interval}{Interval}} r;
  r = set.getInterval(i);
  ... work with r ...
\}
}\end{DoxyPre}


The current implementation does not allocate any dynamic (heap) memory.

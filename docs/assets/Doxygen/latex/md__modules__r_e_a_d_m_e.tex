ICAROUS modules encapsulate the core functionalities required to enable autonomous operations of unmanned systems.\hypertarget{md__modules__r_e_a_d_m_e_autotoc_md15}{}\doxysection{ACCo\+RD}\label{md__modules__r_e_a_d_m_e_autotoc_md15}
A library developed at NASA Langley for airspace applications. ACCo\+RD contains several useful utilities\+:
\begin{DoxyItemize}
\item Data structures for representing aircraft state data (position, velocity, attitude, etc) in various coordinate frames and units.
\item Data structures to encode EUTL flightplans and manipulations on flightplans.
\item Functions to perform common transformations on aircraft data.
\item Geofence functionality (Poly\+CARP)
\item Sense and avoid functionality (DAIDALUS)
\end{DoxyItemize}\hypertarget{md__modules__r_e_a_d_m_e_autotoc_md16}{}\doxysection{Core}\label{md__modules__r_e_a_d_m_e_autotoc_md16}
Suite of libraries implementing autonomous decision making, path planning, traffic avoidance and guidance\+:
\begin{DoxyItemize}
\item Event\+Manager\+: A template library implementing an event handling framework.
\item \mbox{\hyperlink{class_cognition}{Cognition}}\+: Decision making functionality impelemented using the Event\+Manager.
\item \mbox{\hyperlink{class_traffic_monitor}{Traffic\+Monitor}}\+: Traffic avoidance capability (default implementation uses DAIDALUS). Extensible to other sense and avoid tools (e.\+g. ACAS-\/X).
\item Trajectory\+Mananger\+: Path planning capability and trajectory monitoring capability with respect to other intruders and geofences.
\item \mbox{\hyperlink{class_guidance}{Guidance}}\+: Trajectory following functionality.
\item \mbox{\hyperlink{class_merger}{Merger}}\+: Distributed merging functionality.
\item \mbox{\hyperlink{class_geofence_monitor}{Geofence\+Monitor}}\+: Geofence monitoring functionality.
\item \mbox{\hyperlink{class_target_tracker}{Target\+Tracker}}\+: Target tracking and fusion capability (requires GNU Scientific Library -\/ GSL).
\item Utils\+: Helper functions.
\item Interfaces\+: Datastructures shared among all the core modules
\end{DoxyItemize}\hypertarget{md__modules__r_e_a_d_m_e_autotoc_md17}{}\doxysection{Compilation}\label{md__modules__r_e_a_d_m_e_autotoc_md17}

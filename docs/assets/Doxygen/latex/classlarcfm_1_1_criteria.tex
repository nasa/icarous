\doxysubsection{Detailed Description}
Determines if a given velocity vector is within the implicit coordination criteria, both vertically and horizontally.

If a resolution velocity v\textquotesingle{} only changes vertical speed, it is recommended that v\textquotesingle{} be constructed using \mbox{\hyperlink{classlarcfm_1_1_velocity_ad2268d9a4194c81a7c28ff9bc23fd94d}{Velocity.\+make\+Vxyz()}}. \mbox{\hyperlink{classlarcfm_1_1_velocity_a937cfc066fbdcc5566fec28a346482dc}{Velocity.\+make\+Trk\+Gs\+Vs()}} may introduce obvious numeric instability in this case.

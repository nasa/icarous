\doxysubsection{Detailed Description}
A \char`\"{}general state\char`\"{} object that holds Euclidean information about position and velocity for an object. Currently this can either be a (point-\/mass) aircraft or a (possible morphing) polygon. This is intended to be traffic information for a General\+Detector object. ~\newline

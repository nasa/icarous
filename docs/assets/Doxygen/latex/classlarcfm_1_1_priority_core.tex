\doxysubsection{Detailed Description}
This is a collection of functions to determine if one aircraft has priority over another aircraft. It is an encapsulation of the \char`\"{}right of way\char`\"{} rules.

All values are in internal units.

Disclaimer\+: The current rules have been verified to have certain safety and liveness properties, but they have not been validated, that is, they may not behave \char`\"{}naturally.\char`\"{} As we learn more about what is expected, these rules may evolve in the future.

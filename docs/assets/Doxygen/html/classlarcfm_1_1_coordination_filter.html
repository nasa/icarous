Detailed Description

This class implements a simple filter to an input in {-1,1} that guarantees a stable output for at least "n" iterations. This number of iterations is called "tolerance". The tolerance is also the maximum delay for any given input. The filter has the property of being symmetric, i.e., it produces opposite outputs to opposite inputs. All filters for a given signal should have the same tolerance and they should be reset at the same time.
